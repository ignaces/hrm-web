## Visual Studio Code
Nuestra Herramienta por defecto para editar código es Visual Studio Code. Es software libre y tiene muchas herramientas para facilitar la pega.

Descarga [Visual Studio code](https://code.visualstudio.com/download)

## Adicionales
Para ayudarnos más aun podemos instalar las siguientes extensiones haciendo click en ![logo](_media/extensionesvs.png) a la izquierda de la vista :

 * Material Icon Theme
 * Edge template support
